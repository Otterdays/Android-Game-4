## Android Development
- **One-Click Deploy:** Connect your Android device via USB and click the Android icon in the top right of the Godot Editor.
- **Remote Debugging:** Enable `Deploy with Remote Debug` in the `Project -> Export` menu to see GDScript errors and prints from your device in the editor console.
- **Logcat:** Use `adb logcat -s godot` to view engine-level logs from the device.
